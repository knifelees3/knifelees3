而该函数的mode字，有以下选项，

Character Meaning
r 读（默认）
w open for writing, truncating the file first
x 打开以进行独占创建，如果文件已经存在则失败
a 打开进行写入，如果存在则追加到文件末尾
b 二进制模式
t 文字模式（默认）
+ 打开进行更新（读写）
