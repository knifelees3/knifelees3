逐个安装软件

默认的Lumerical FDTD Solutions为rpm文件, 可以直接通过rpm install <packagename.rpm> 安装，之前Ubuntu还需要自己手动转为deb包，这一点方便了许多。不过安装时一般都会有库缺少问题，CentOS的yum支持自动安装依赖，所以只需要cd到对应的文件夹，依次安装即可, 比如安装FDTD模块，
